The Interactive Media Design program has taught me many new skills such as coding using HTML and CSS, editing, and altering photos using Lightroom and Photoshop, creating videos using Premiere Pro, re-designing and creating website mockups using Adobe XD, and much more. I am looking forward to the many new skills I will be picking up along the way.
